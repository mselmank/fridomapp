# fridomapp

📦  #### Run Locally ####📦
                                                              
1- ## Clone repo

    $ git clone https://github.com/mselmank/fridomapp.git

    $ cd frido
    
    $ cd fridoapp
    
          
2- ## Npm install 🚀          
          
        # open new terminal
        $ cd frido
        $ cd fridoapp
        $ npm install 
        $ npm run dev  
          
          
3- ## Open the app at the URL:

       # http://localhost:3000
